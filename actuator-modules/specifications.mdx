## Overview

This page provides complete technical specifications for the Brushless Actuator Module Core. Specifications apply to both [v1.0](/actuator-modules/core-v1) and [v1.1](/actuator-modules/core-v1-1) modules, which are functionally identical.

## Performance Specifications

  <Card title="Torque Output" icon="gauge-high">

    Peak torque higher depending on thermal limits
  </Card>
  <Card title="Gear Ratio" icon="gears">

  </Card>

## Encoder Specifications

### Broadcom AEDT-9810-Z00

| Parameter | Specification |
|-----------|---------------|
| **Type** | Optical incremental encoder |
| **Resolution** | 5,000 pulses/revolution per channel |
| **Counts** | 20,000 counts/rev (quadrature) |
| **Output Type** | Two-channel quadrature + index (ABZ) |

  **Accuracy**: The high 5,000 cpr resolution provides excellent position accuracy for torque control applications.
</Info>

### Timing Belt Details

**Conti Synchroflex AT3 GEN III**

- **Material**: Heavy-duty polyurethane with steel cord reinforcement
- **Profile**: AT3 (3mm pitch)
- **Generation**: GEN III (latest version)
- **Total Weight**: 4.5g both belts
- **Manufacturer**: Continental (Contitech)

<Warning>
  **Custom Tooth Profile**: Pulley tooth profile is modified from standard AT3 to reduce backlash. Do not substitute with standard AT3 pulleys.
</Warning>
